# References and Borrowing

- References allow us to make a reference to a value without taking ownership of it
    - Borrowing the value

## Two types of references
- Shared
    - You can read but not modify whatever is being referenced
    - You can have multiple shared references to a value

- Mutable
    - Allow you to read and modify the value
    - You can't have any other references of any type to that value at the same time
