# Installing Rust

## Mac

- Run the following command in the Terminal to install Rust:
- `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
- [Additional Installation Instructions](https://www.rust-lang.org/tools/install)
