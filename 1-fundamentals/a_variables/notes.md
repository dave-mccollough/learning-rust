# Variables

- Variables store data
- Mutability refers to how that data can be manipulated
    - Mutable data can be modified
    - Immutable data can't be modified
    - Add `mut` keyword to make a variable mutable
        - `let mut x = 5`

## `let` Keyword
- The `let` keyword allows you to assign variables to the current scope
    - `let x = 5`
- Variable tyoe can be be declared, but isn't required

## `const` Keyword
- The `const` keyword allows you to assign variables in any scope (including global)
    - `const WORD: String = "Hello World"`
- Variable tyoe must be declared
- `const` variable names are in ALL CAPS
