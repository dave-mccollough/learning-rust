# Structs

- Short for structures
- Custom data type for grouping related data
- Create a `struct` by using the struct keyword then the name of the struct
- Use camel case
- Structs are key/pair values

## Three Types of Structs
- Name fields
    - Gives a name to each component in a struct
- Tuples
    -  Tuple structs allow you to group multiple values together without naming them.
    - Identifies them by the order they appear
- Unit
    - No components
